# UUID to Hex 변환기

이 프로젝트는 UUID를 16진수로 변환하는 간단한 GUI 애플리케이션입니다. Electron을 사용하여 구축되었으며, 크로스 플랫폼에서 실행될 수 있습니다.

## 기능

- UUID 입력 및 16진수로 변환
- 변환된 결과를 클립보드에 복사

## 설치

1. 이 저장소를 클론합니다:

   ```
   git clone https://github.com/yourusername/uuid-converter.git
   cd uuid-converter
   ```

2. 의존성을 설치합니다:
   ```
   npm install
   ```

## 실행

개발 모드에서 애플리케이션을 실행하려면:

```
npm run start
```
